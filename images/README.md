# 📸 Carpeta de Imágenes

Esta carpeta contiene todas las imágenes necesarias para la landing page.

## 🎯 Imágenes Necesarias

### 1. Categorías (4-6 imágenes)

**Nombres sugeridos**: `categoria-1.jpg`, `categoria-2.jpg`, etc.

**Dimensiones recomendadas**: 
- Ancho: 800px
- Alto: 600px
- Ratio: 4:3
- Formato: JPG o WebP
- Peso: < 200KB cada una

**Contenido**: 
- Fotos de productos
- Fotos de procesos
- Ambientes de trabajo
- Productos en uso

---

### 2. Iconos de Beneficios (opcional)

**Nombres sugeridos**: `icono-beneficio-1.png`, etc.

**Dimensiones**: 
- 200x200px
- Formato: PNG con fondo transparente
- Peso: < 50KB

**Alternativa**: Puedes usar emojis (configurado por defecto en `config.js`)

---

## 🛠️ Herramientas Recomendadas

### Para Optimizar Imágenes:

1. **TinyPNG** - https://tinypng.com/
   - Comprime JPG/PNG sin perder calidad
   - Gratis hasta 20 imágenes a la vez

2. **Squoosh** - https://squoosh.app/
   - Compresión avanzada
   - Conversión a WebP

3. **Photopea** - https://www.photopea.com/
   - Photoshop online gratis
   - Para redimensionar y editar

### Para Convertir a WebP:

WebP es un formato moderno que pesa 30% menos que JPG.

```bash
# Con ImageMagick (si lo tienes instalado)
magick convert imagen.jpg imagen.webp
```

O usa **Squoosh** (más fácil).

## ✅ Checklist de Imágenes

Antes de publicar, asegúrate de:

- [ ] Todas las imágenes están optimizadas (< 200KB)
- [ ] Los nombres de archivo son descriptivos (sin espacios ni tildes)
- [ ] Las imágenes tienen buena calidad visual
- [ ] Todas las rutas en `config.js` están correctas
- [ ] Las imágenes se ven bien en móvil

---

## 🎨 Tips de Fotografía

### Para Productos:
- Fondo blanco o neutro
- Buena iluminación natural
- Múltiples ángulos
- Alta resolución (mínimo 1200px de ancho)

### Para Ambientes:
- Evitar fotos de stock genéricas
- Usar fotos reales del negocio
- Mostrar personas si es posible (más confianza)
- Evitar fotos borrosas o pixeladas
